VWorld 서비스 URL 등록용 기본 페이지

이 페이지는 VWorld에 서비스 URL로 등록할 기본 엔트리입니다. 지금은 최소 구조만 두었고, 이후 Blender에서 만든 3D 모델과 VWorld API 호출 코드를 여기에 연결하면 됩니다.

기본 구조

  • 서비스 URL 엔트리준비됨
  • VWorld API 연결 지점대기
  • Blender 3D 모델 연결 지점대기

등록 URL 확인

VWorld에 등록하는 서비스 URL은 실제 배포 주소와 정확히 일치해야 합니다. 프로토콜, 도메인, 포트, 경로가 다르면 인증 또는 API 호출 오류가 날 수 있습니다.
