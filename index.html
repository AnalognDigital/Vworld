VWorld Service URL Entry

VWorld 서비스 URL 등록용 기본 페이지

이 페이지는 VWorld에 서비스 URL로 등록할 기본 엔트리입니다. 지금은 최소 구조만 두었고, 이후 Blender에서 만든 3D 모델과 VWorld API 호출 코드를 여기에 연결하면 됩니다.

기본 구조

  • 서비스 URL 엔트리준비됨
  • VWorld 3D API 연결준비됨
  • Blender 3D 모델 연결 지점다음 단계

등록 URL 확인

VWorld에 등록하는 서비스 URL은 실제 배포 주소와 정확히 일치해야 합니다. 프로토콜, 도메인, 포트, 경로가 다르면 인증 또는 API 호출 오류가 날 수 있습니다.

현재 URL 확인 중...

예시: `https://your-domain.com/` 또는 `https://your-domain.com/vworld/`

API Key

확인 중...

브이월드 WebGL 3D API 3.0은 브라우저에서 사용할 때 `apiKey`와 `domain`을 함께 전달해야 합니다. 이 페이지는 로컬 `api.env` 또는 배포 시 생성되는 `static/config.js`를 통해 키를 읽습니다.

GitHub Pages에서는 `api.env`를 직접 읽을 수 없으므로 GitHub Secret에서 `static/config.js`를 생성해 사용합니다.
